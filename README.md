hoa?hoa! application server입니다. API, Database 등 'backend' 작업 정보들로 구성되어 있습니다.

이 project는 javascript로 되어 있고, Node.js, Express를 이용하였습니다. Database는 Sequelize를 이용하여 MySQL을 사용하였습니다. 이 Server는 개발, 서버 on/off 가 가능합니다.

- AWS

  - EC2

- Express
